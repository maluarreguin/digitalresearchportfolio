Core Seminar

Final Project
Created for Digital Humanities 201
Winter 2023

Link to Final Project

This project was created for my core course in Digital Humanities. For this assignment, I was tasked to create a digital project that would allow me to expand upon my chosen research topic while utilzing the various skillsets we learned within the course inclduing website building, data visualizations, and maps. My intention behind this project was to delve into one speific aspect I found to be intersting regarding the database of The Survey of Scottish Witchcraft, and in turn, this avenue was to be that of fairies. I approached this project as a form of exploration rather than definitives, as I feel as though I lack the histoical knowledge to make longstanding claims from my one dataset. However, the dataset is incredibly thorough, and does an excellent job at structuring and organizing these cases into the greater context of the witchcraft trials at the time. Further, I believe what we can grasp from the data I have utilized for this project is what this smaller subsection highlights about this period at large.
